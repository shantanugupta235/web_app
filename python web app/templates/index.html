Armstrong Number

Armstrong Number : A positive number is called armstrong number if it is equal to the sum of cubes of its digits.

Example : 153 is an Armstrong number because 1*1*1 + 5*5*5 + 3*3*3 = 153

120 is not a Armstrong number. 1*1*1 + 2*2*2 + 0*0*0 = 9
